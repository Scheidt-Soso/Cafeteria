# Sistema-de-Gest-o-para-Cafeteria-
Sistema de gestão para cafeteria desenvolvido em Java com integração ao MySQL utilizando JDBC. O sistema permitirá o cadastro de clientes, produtos e pedidos, controle de estoque, processamento assíncrono com threads e geração de relatórios SQL, aplicando conceitos de POO, SOLID e versionamento com Git.
